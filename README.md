# Medical Data Analysis Using SQL

## Overview
This project focuses on analyzing a medical dataset using SQL to extract meaningful insights. It involves writing and executing multiple complex queries to explore patterns, trends, and relationships within the data.

## Objective
To develop strong SQL querying skills by solving real-world analytical problems using structured medical data and generating insights through advanced query techniques.

## Tools & Technologies
- SQL (Data querying and analysis)
- Microsoft PowerPoint (Presentation of queries and results)

## Key Analysis Performed
- Data filtering, aggregation, and grouping
- Use of joins to combine multiple tables
- Subqueries and nested queries for deeper analysis
- Window functions and advanced SQL logic
- Conditional queries for targeted insights

## Deliverables
- Collection of 30+ SQL queries covering basic to advanced scenarios
- PowerPoint presentation showcasing queries and their outputs

## Skills Demonstrated
- SQL querying (basic to advanced)
- Data analysis and interpretation
- Problem-solving using structured data
